# Hotel Booking Cancellation Analysis

## Project Overview
This project analyzes hotel booking cancellation patterns for StayWell Hotels Group 
using Power BI. The analysis covers cancellation trends, key drivers, and 
segment risk analysis across 119,390 bookings.

## Key Findings
- Overall cancellation rate: 37.04%
- Total cancellations: 44K bookings
- Total revenue lost: $17M
- Groups segment has highest cancellation rate at 61%
- Non-refundable deposits show 99.4% cancellation rate (data anomaly flagged)
